KOA3 server README

📙 このウェブアプリケーションについて

  • リモートコンピュータとのファイルの送受信を行う。
  • リモートコンピュータのファイルシステムの操作を行う。

📙 インストール

このプロジェクトを Github からクローンして、npm install を実行して必要なモジュールをインストールする。

$ git clone https://gihub.com/makandat/koa3_server

📙 起動

node index.js または npm start で起動できる。

ファイルアップロード

エクスプローラなどからアップロードするファイルを「ここにファイルをドロップ」領域にドロップして「今すぐアップロード」ボタンをクリックするとファイルアップロードが開始される。

ファイルダウンロード

メインページでダウンロードしたいファイルの左にある * をクリックするとそのファイルのパスがコピーされるので、「ファイルダウンロード」テキストエリアに貼り付ける。 「ダウンロードリンクを作成」ボタンをクリックするとダウンロードリンクが作成されるので、そのリンクをクリックするとダウンロードが開始される。

場所のクイックアクセス

メイン画面の上の方にはよく使う場所のリンクがある。これは、「場所の編集」メニューで編集できる。

フォルダの作成

「フォルダ作成」メニューを実行すると、フォルダ作成ページが表示されるのでフォルダのフルパスを入力して「作成する」ボタンをクリックする。

ファイルやフォルダの削除

ファイルを削除するには「ファイル削除」メニューを、フォルダを削除するには「フォルダ削除」メニューを実行する。これらのメニューを実行するとそれぞれのページが開くので、フルパスを入力して「削除する」ボタンをクリックする。
注意：フォルダ削除の機能は Linux コマンドを実行して削除を行うため、Windows では使用できない。

テキストファイルのプレビュー

ファイル一覧表示で、ファイルをクリックすると右側領域にあるテキストファイル・プレビューにそのファイル内容が表示される。バイナリーファイルの区別はしないので、拡張子などで判別してバイナリーファイルを表示しないように注意すること。

タールボール (.tar.gz) の作成

フォルダをまとめてダウンロードするためには、まず .tar.gz ファイルを作成する必要がある。「タールボール作成」メニューを実行すると、タールボール作成ページが開く。

タールボール (.tar.gz) の解凍

フォルダをまとめてアップロードするためには、クライアント側で .tar.gz ファイルを作成してファイルアップロードする。「タールボール解凍」メニューをクリックすると、タールボール解凍のためのページが開く。

ファイルのコピー

「ファイルコピー」メニューを実行すると、ファイルコピーのページが開く。「複数ファイルのコピー」メニューを実行すると、複数ファイルのコピーページが開く。

名前の変更

「名前の変更」メニューを開くと名前の変更をするためのページが開く。

コマンド実行

「コマンド実行」メニューをクリックすると、コマンドを実行するためのページが開く。

ファイルモードの変更

「ファイルモードの変更」メニューをクリックすると、ファイルモードを変更するためのページが開く。(注意) Windows は不可。
